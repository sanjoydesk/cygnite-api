The Default Controller -------------------------------------------------------------------------- This controller respond to uri begining with welcomeuser and also respond to root url like "welcomeuser/index"

Your GET request of "welcomeuser/form" will repond like below -
public function action_form() { echo "Cygnite : Hellow ! World "; } Note: By default cygnite doesn't allow you to pass query string in url, which consider as bad url format.

You can also pass parameters into the function as below- Your request to "welcomeuser/form/2134" will pass to
public function action_form($id = ") { echo "Cygnite : Your user Id is $id"; } In case if you are not able to access parameters passed into method directly as above, you can also get the uri segment echo Url::segment(2);

That's it you are ready to start your awesome application with Cygnite Framework.
